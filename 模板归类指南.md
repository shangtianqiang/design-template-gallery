## 快速选择指南

### 按色调偏好选

| 想要的感觉 | 推荐模板 |
|-----------|---------|
| **温暖人文** | Claude、Airbnb、PostHog、Mistral AI、Replicate |
| **暗黑克制** | Linear、Vercel、Spotify、x.ai、ClickHouse |
| **黑白极简** | Tesla、Nike、SpaceX、Bugatti、Apple |
| **粉彩活泼** | Figma、Notion、Miro、Slack、ElevenLabs |
| **渐变大气** | Stripe、Together AI、Framer、Mintlify |
| **单色点缀** | Supabase（绿）、Cursor（橙）、VoltAgent（绿）、Wise（绿） |

### 按目标用户选

| 目标用户 | 推荐模板 |
|---------|---------|
| **开发者/工程师** | Linear、Vercel、Cursor、Warp、Raycast、Supabase |
| **设计师/创意人** | Figma、Framer、Webflow、RunwayML、ElevenLabs |
| **企业/B2B 决策者** | IBM、Cohere、Stripe、Intercom、Airtable |
| **普通消费者** | Airbnb、Spotify、Shopify、Nike、Starbucks |
| **金融/投资用户** | Stripe、Coinbase、Revolut、Wise、Kraken |

### 按产品类型选

| 产品类型 | 推荐模板 |
|---------|---------|
| **SaaS 营销页** | Stripe、Notion、Zapier、Intercom |
| **开发者文档** | Mintlify、OpenCode、Ollama |
| **电商/商城** | Shopify、Nike、Airbnb、Meta |
| **AI 产品** | Claude、Cursor、ElevenLabs、Lovable |
| **展示型官网** | Apple、Tesla、Ferrari、SpaceX |
| **数据/分析** | PostHog、ClickHouse、MongoDB |
