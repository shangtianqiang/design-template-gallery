# Design Template Gallery

一个可视化的设计模板画廊，基于 [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) 项目提供的 DESIGN.md 文件构建。

**[>>> 打开画廊 <<<](https://shangtianqiang.github.io/design-template-gallery/)**

## 项目简介

这是一个交互式的 Web 画廊，用于浏览和预览各种网站的 DESIGN.md 设计模板。每个模板都包含了完整的设计系统信息，包括颜色、字体、组件样式等。

## 功能特性

- **可视化预览**：每个模板都以卡片形式展示，包含颜色预览、字体示例和组件样式
- **分类筛选**：支持按类别筛选模板
  - Developer Tools
  - AI & LLM
  - Fintech
  - E-commerce
  - Productivity
  - Design Tools
  - 等等...
- **搜索功能**：支持按名称搜索模板
- **详细查看**：点击卡片可查看完整的设计系统详情
- **一键复制**：支持复制 DESIGN.md 内容到剪贴板

## 使用方法

1. 访问 [画廊页面](https://shangtianqiang.github.io/design-template-gallery/)
2. 使用左侧的分类筛选或搜索框查找模板
3. 点击卡片查看模板详情
4. 复制 DESIGN.md 内容到你的项目中
5. 告诉你的 AI 代理使用该设计系统

## 技术实现

- 纯 HTML/CSS/JavaScript 实现，无需构建工具
- 响应式设计，支持桌面和移动设备
- 使用 CSS Grid 布局展示模板卡片
- 支持深色主题
